# 我的个人小站

这是一个可部署为公开网页的静态项目，适合做一个简洁的个人主页、作品页或小玩意页面。

## 文件说明

- `index.html` - 页面入口
- `styles.css` - 页面样式
- `script.js` - 按钮交互与语音效果

## 本地预览

直接在浏览器中打开 `index.html` 即可。

## 发布成可分享的网址

最简单的方式是把这个项目上传到 GitHub，然后用 GitHub Pages 或 Vercel / Netlify 发布。

### 方案 1：GitHub Pages（免费，最适合你）

1. 把这个项目上传到 GitHub 仓库。
2. 打开仓库页面，进入 Settings > Pages。
3. 选择部署来源为 `Deploy from a branch`。
4. 选择主分支（如 `main`）和根目录。
5. 保存后，等待几分钟。
6. 你就会得到一个公开链接，例如：
   `https://你的用户名.github.io/你的仓库名/`

### 方案 2：Vercel / Netlify（更省心）

1. 把项目上传到 GitHub。
2. 在 Vercel 或 Netlify 中导入这个仓库。
3. 默认会自动生成一个公开链接。
4. 以后你可以直接分享这个链接给朋友。

### 以后绑定自己的域名

如果你想要更像“属于你自己的地址”，可以后续购买一个域名，例如：
- `yourname.com`
- `www.yourname.com`

然后在 GitHub Pages / Vercel / Netlify 的设置里绑定这个域名即可。
